Welcome to Young-Genius Game!

For whom

This is a simple game for 6-10-year-old children or kitties.

How much

The game is free. But if you like, you can donate the author. She'll buy some programming books for your donations.

System requirements

The latest version of Google Chrome should be installed. You'll need also some working computer for that, preferably with a mouse, keyboard and screen.

Display Resolution

800 × 600 pixels or sup.

Game rules

The first rule: there is no rules.

The second rule: break the rules and win!

Game controls

Sortable task

Tab - Next Item

Shift+Tab - Previous Item

Up Arrow / Left Arrow - Move item left

Down Arrow / Right Arrow - Move item right

"h" / Home - Move item to front

"e" / End - Move item to end

"r" - Reverse the list order

"s" - Shuffle the list

"Enter" - submit

Menu

Alt - open menu

Esc - close menu

Use Up and Down arrows to navigate

Task Choosing

Space - open tasks

Esc - close tasks

Use Up and Down arrows to navigate

Answer Window

Tab - Next Item

Shift+Tab - Previous Item

Enter - submit your answer

(Tab should be on submit button)

Form Window

Tab - Next Item

Shift+Tab - Previous Item

Enter - submit and launch the game

(Tab should be on submit button)
