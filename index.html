Welcome to Young-Genius Game!

For whom

This is a simple game for 6-10-year-old children or kitties.

How much

The game is free. But if you like, you can donate the author. She'll buy some programming books for your donations.

System requirements

The latest version of Google Chrome should be installed. You'll need also some working computer for that, preferably with a mouse, keyboard and screen.

Display Resolution

800 × 600 pixels or sup.

Game rules

The first rule: there is no rules. The second rule: break the rules and win!
